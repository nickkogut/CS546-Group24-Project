# CS 546 - Group 24 Final Project - Career Scope NYC 

## Seed instructions:

1. Unzip tasks/seed_data.zip and put its contents into tasks/ (If it was unzipped into a new folder, make sure to remove it from that folder and place it directly here)
2. With a mongo server running, do "npm run seed"

## App running instructions

1. Open a terminal in the project's root directory
2. npm i
3. npm start
4. Connect at localhost:3000/
